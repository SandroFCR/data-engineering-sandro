# Sincronizar Laptop

Repositorio remoto:

```text
https://github.com/SandroFCR/data-engineering-sandro.git
```

## Clonar En Otra Maquina

```powershell
git clone https://github.com/SandroFCR/data-engineering-sandro.git
cd data-engineering-sandro
```

## Actualizar Cambios

Antes de trabajar:

```powershell
git pull
```

Despues de trabajar:

```powershell
git status
git add .
git commit -m "Describir cambio"
git push
```

## Estructura Importante

- `clases/`: notas y dinamicas de aprendizaje.
- `proyectos/`: proyectos de portafolio.
- `proyectos/01_crypto_etl_pipeline/`: primer pipeline ETL.
