# Semana 01 - Fundamentos Para El Primer Pipeline

## Objetivo

Construir tu primer pipeline simple y entender el flujo basico de la Ingenieria de Datos.

## Conceptos

- Que es una API.
- Que es JSON.
- Que es ETL.
- Que es una base de datos.
- Diferencia entre datos raw y datos limpios.
- Por que SQL es clave.

## Plan De Estudio

### Dia 1

- Entender el proyecto.
- Preparar estructura de carpetas.
- Elegir fuente de datos.

### Dia 2

- Consumir una API con Python.
- Guardar respuesta JSON raw.

### Dia 3

- Leer JSON.
- Transformarlo a tabla.
- Guardarlo como CSV.

### Dia 4

- Crear base SQLite.
- Cargar datos limpios.

### Dia 5

- Consultar con SQL.
- Responder preguntas de negocio.

### Dia 6

- Mejorar el pipeline.
- Agregar logs y manejo de errores.

### Dia 7

- Documentar el proyecto.
- Hacer mini retrospectiva.

## Reto De La Semana

Explicar con tus palabras este flujo:

```text
extraer -> transformar -> cargar -> consultar -> decidir
```
